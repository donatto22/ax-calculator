ES modules

You can import everything, such as only the object you need.

                    
                        import * as ax from 'ax-calculator'
                    
                
                    
                        import { ... } from 'ax-calculator'
Browser

You will need an external file that imports the calculator, and specify that it is of module type.

                        
                        <!DOCTYPE HTML>
                        <html>
                        <head>
                            <script type="module" src="file.js"></script>
                        </head>
                            <body>
                            </body>
                            </html>
Usage

The code below is a small demonstration of the calculator. If you want to see the complete documentation visit here.

                    
                        const chaining = new BasicOperations.Chaining()
                        console.log(chaining.baseNumber(5).multiply(4).divide(2).value); // 10
                        
                        var array_x = [100, 90, 80, 45, 50, 50, 60, 40, 25, 20];
                        var array_y = [3, 5, 9, 10, 20, 21, 24, 24, 27, 35];
